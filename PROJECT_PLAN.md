# Study Agent 项目规划

> 当前活跃阶段：`v0.7.0`  
> 当前状态：微信群联网搜索、页面文本增强、来源追溯、打包安全与状态稳定性已进入联调阶段。

## 当前定位

当前项目已经不再只是“学习对话 + 群聊反馈”，而是进入：

```text
学习对话
+ 课后更新
+ 微信群互动
+ 轻量联网资讯检索
+ 页面文本增强摘要
```

## v0.7.0 已完成的重点

1. 微信群支持用户输入查询词进行联网检索
2. 搜索支持多源 RSS 聚合
3. 搜索可尝试读取前 5 条页面文本
4. 搜索结果最多保留 10 条新闻
5. 来源块写入群聊记录
6. 页面读取失败自动降级
7. URL 安全边界增强
8. 页面编码解码增强
9. 打包排除替换目录与非运行产物

## 当前仍在验证的点

1. `pytest` 全量回归
2. 搜索速度是否可接受
3. 页面文本质量是否稳定
4. 多源聚合是否会引入太多噪声
5. “聊最近新闻”是否需要进一步轻量化

## v0.7.1 目标

下一阶段优先级：

1. 搜索来源 UI 展示优化
2. 链接点击体验
3. 搜索按钮 cooldown / running lock 增强
4. 页面文本质量进一步优化
5. 测试环境与真实运行目录隔离

## 文档分工

1. `README_v0_7_0.md`: 当前版本说明
2. `USER_GUIDE.md`: 当前使用方法
3. `FUTURE.md`: 下一阶段方向
