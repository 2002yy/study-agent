# 个人学习 Agent 系统

## 当前阶段

v0.7.3 - 服务层拆分与工程化收口，wechat_service 拆分，CI 落地

## 核心目标

支持长期学习、项目推进、论文修改、课后复盘，以及更自然的多角色微信群互动。

## 当前能力

- 角色化学习伙伴：三月七、刻晴、纳西妲、流萤
- 单聊 / 微信群双入口
- `flash / pro` 模型切换
- `fast / standard / deep` 性能模式
- 长期记忆读取与受控写入
- 课后更新、微信群反馈、记忆候选提取
- 群聊流式互动与四角色回复归一
- 微信群联网搜索、多源新闻聚合、正文尝试读取与来源块写回

## 当前重点

- 让微信群联网搜索在“最新、可信、可追溯”三个维度更稳
- 保持左侧栏状态、memory 文本和打包文档一致
- 收口来源展示与摘要边界，避免旧新闻或标题线索误导用户
